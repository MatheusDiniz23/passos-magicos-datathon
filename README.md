# Datathon Passos Mágicos: Análise de Dados Educacionais e Modelo Preditivo

## Descrição do Projeto
Este projeto faz parte de um Datathon de Pós-Graduação em Análise de Dados, focado na análise de dados educacionais da ONG 'Passos Mágicos'. O objetivo é identificar indicadores de desenvolvimento estudantil e construir um modelo preditivo para identificar alunos em risco de lacunas de aprendizado acadêmico.

## Descrição dos Datasets
Os datasets utilizados neste projeto são:
- `BASE DE DADOS PEDE 2024 - DATATHON.xlsx`: Contém dados educacionais para o Datathon.
- `PEDE_PASSOS_DATASET_FIAP.csv`: Contém dados adicionais para o projeto.

## Objetivos do Projeto
- Analisar indicadores de desenvolvimento estudantil.
- Construir um modelo preditivo para identificar alunos em risco de lacunas de aprendizado acadêmico.
